# Want to self-host the bot?

- **Clone the repo:**

  - **Using CLI:** `gh repo clone TRaya1n/qwik`
  - **Using HTTPS:** `https://github.com/TRaya1n/qwik.git`

- **Config .env file:**

  - `/.env.sample`

- **Start:**
  - `npm run start`
